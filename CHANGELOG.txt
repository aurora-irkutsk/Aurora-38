🔄 КАК ВКЛЮЧИТЬ CURSOR TRAIL ОБРАТНО

Откройте файл: advanced-animations.js

Найдите строки (около строки 594):
  // 1. Cursor Trail (только для десктопа) - ОТКЛЮЧЕНО
  // if (!isMobile) {
  //     new CursorTrail();
  //     console.log('✓ Cursor Trail активирован');
  // }
  console.log('⚠️ Cursor Trail отключен');

Замените на:
  // 1. Cursor Trail (только для десктопа)
  if (!isMobile) {
      new CursorTrail();
      console.log('✓ Cursor Trail активирован');
  }

Сохраните и обновите страницу в браузере (Ctrl+F5)
